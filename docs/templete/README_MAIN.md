# UniGal-EnforcementProposal

UniGal-EnforcementProposal，简称UEP，是UniGal的警告和错误汇总形成文档的提案。

## 处理方式

### Warning

Warning是因为不同引擎之间存在差异

### Error

Error将终止编译器的运行
